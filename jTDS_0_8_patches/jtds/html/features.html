jTDS feature matrix

Below is the list of features of jTDS release 0.8-rc1.

Compliance
SQL Server 2000, 7.0, 6.5 Compliant Yes
Sybase Compliant Yes (not all features available for Sybase)
JDK 1.4, 1.3 and 1.2 Compliant Yes
Sun Certified 100% Pure JavaTM Yes

Feature Summary
Open Source and Freely Distributable Yes
Prepared and Callable Statements Yes
Updateable Result Sets Yes
Scrolling Result Sets Yes
Batch Updates Yes; not completely optimized, but fully functional and still faster than most drivers
Statement Pooling Yes
Full Timeout Support Yes
Full Range of Cursor Types Yes
Full BLOB and CLOB Support Yes; OutOfMemoryErrors can occur with large LOBs; scheduled for fixing
Unlimited Statements per Connection Yes
Generated keys retrieval Yes

J2EE Support
JNDI Data Sources Yes
Connection Pooling Yes
XA Distributed Transactions No

SQL Server-Specific
Optimized Native Protocol Performance Yes; fastest JDBC driver for SQL Server (benchmarks results: [1], [2])
Windows Authentication Yes
SQL Server named instances Yes
Ignoring of update counts from triggers Yes
Send String data as ASCII/Unicode Yes
NTEXT, NCHAR and NVARCHAR Support Yes
