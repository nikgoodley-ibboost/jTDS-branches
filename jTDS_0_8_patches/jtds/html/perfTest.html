NetDirect JDBC Performance Test Results

These are the results of running JNetDirect's JDBC Performance Benchmark. The tests were run against an SQL Server 2000 over a 100Mbit network connection. The client machine was an AMD Duron 900MHz with 512MB RAM (no idea about the server).

4 drivers were tested: i-net Merlia 6.0, Microsoft SQL Server Driver for JDBC SP2, NetDirect JSQLConnect 3.3023 and jTDS 0.7. All tests were run 5 times and the final result is the average of the 5 test runs.

Unfortunately, the license of NetDirect JSQLConnect does not allow any benchmarking results to be made public, but its results would not have affected the final standings too much. If you want to see the results for JSQLConnect, download it and the JDBC Performance Benchmark and do the testing yourself. It's very easy.
